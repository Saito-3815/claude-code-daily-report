### SCR-001: ログイン画面

#### 基本情報

| 項目    | 内容     |
| ------- | -------- |
| 画面 ID | SCR-001  |
| 画面名  | ログイン |
| URL     | /login   |
| 認証    | 不要     |

#### 画面レイアウト

```
+------------------------------------------+
|           営業日報システム                 |
+------------------------------------------+
|                                          |
|        +------------------------+        |
|        | メールアドレス          |        |
|        | [                    ] |        |
|        +------------------------+        |
|                                          |
|        +------------------------+        |
|        | パスワード              |        |
|        | [                    ] |        |
|        +------------------------+        |
|                                          |
|        [      ログイン       ]           |
|                                          |
|        パスワードを忘れた方はこちら        |
|                                          |
+------------------------------------------+
```

#### 入力項目

| No  | 項目名         | 項目 ID  | 種別       | 必須 | 桁数 | 初期値 | 備考               |
| --- | -------------- | -------- | ---------- | ---- | ---- | ------ | ------------------ |
| 1   | メールアドレス | email    | テキスト   | ○    | 255  | -      | メール形式チェック |
| 2   | パスワード     | password | パスワード | ○    | 128  | -      | マスク表示         |

#### ボタン・リンク

| No  | 名称                         | ID          | 種別   | 処理内容                                       |
| --- | ---------------------------- | ----------- | ------ | ---------------------------------------------- |
| 1   | ログイン                     | btn_login   | ボタン | 認証処理を実行し、成功時はダッシュボードへ遷移 |
| 2   | パスワードを忘れた方はこちら | link_forgot | リンク | パスワードリセット画面へ遷移                   |

#### バリデーション

| No  | 対象項目       | チェック内容 | エラーメッセージ                                 |
| --- | -------------- | ------------ | ------------------------------------------------ |
| 1   | メールアドレス | 必須チェック | メールアドレスを入力してください                 |
| 2   | メールアドレス | 形式チェック | 正しいメールアドレス形式で入力してください       |
| 3   | パスワード     | 必須チェック | パスワードを入力してください                     |
| 4   | 全体           | 認証チェック | メールアドレスまたはパスワードが正しくありません |

#### 画面レイアウト

```
+------------------------------------------------------------------+
| [ロゴ]  営業日報システム          山田太郎 ▼  [ログアウト]         |
+------------------------------------------------------------------+
| [ダッシュボード] [部下日報] [顧客マスタ] [営業マスタ]               |
+------------------------------------------------------------------+
|                                                                  |
|  日報作成                                      2024年01月15日    |
|  ================================================================|
|                                                                  |
|  ■ 訪問記録                                      [+ 行を追加]    |
|  +------------------------------------------------------------+  |
|  | No | 顧客名              | 訪問時刻 | 結果     | 訪問内容   |  |
|  +------------------------------------------------------------+  |
|  | 1  | [顧客選択 ▼]        | [09:00]  | [商談中▼]| [       ]  |  |
|  |    |                     |          |          | [削除]     |  |
|  +------------------------------------------------------------+  |
|  | 2  | [顧客選択 ▼]        | [11:00]  | [成約 ▼] | [       ]  |  |
|  |    |                     |          |          | [削除]     |  |
|  +------------------------------------------------------------+  |
|                                                                  |
|  ■ 課題・相談 (Problem)                          [+ 行を追加]    |
|  +------------------------------------------------------------+  |
|  | 1  | [                                              ] [削除] |  |
|  | 2  | [                                              ] [削除] |  |
|  +------------------------------------------------------------+  |
|                                                                  |
|  ■ 明日やること (Plan)                           [+ 行を追加]    |
|  +------------------------------------------------------------+  |
|  | 1  | [                                              ] [削除] |  |
|  +------------------------------------------------------------+  |
|                                                                  |
|        [下書き保存]                    [提出]                    |
|                                                                  |
+------------------------------------------------------------------+
```

#### 入力項目

| No  | 項目名       | 項目 ID         | 種別           | 必須 | 桁数 | 初期値 | 備考                        |
| --- | ------------ | --------------- | -------------- | ---- | ---- | ------ | --------------------------- |
| 1   | 報告日       | report_date     | 日付           | ○    | -    | 当日   | 変更不可                    |
| 2   | 顧客名       | customer_id     | セレクト       | ○    | -    | -      | 顧客マスタから選択          |
| 3   | 訪問時刻     | visit_time      | 時刻           | -    | -    | -      | HH:MM 形式                  |
| 4   | 結果         | result          | セレクト       | -    | -    | -      | 商談中/成約/見送り/情報収集 |
| 5   | 訪問内容     | content         | テキストエリア | ○    | 2000 | -      | 複数行入力可                |
| 6   | 課題・相談   | problem_content | テキストエリア | -    | 2000 | -      | 複数行入力可                |
| 7   | 明日やること | plan_content    | テキストエリア | -    | 2000 | -      | 複数行入力可                |

#### ボタン・リンク

| No  | 名称                 | ID              | 種別   | 処理内容                               |
| --- | -------------------- | --------------- | ------ | -------------------------------------- |
| 1   | 行を追加（訪問記録） | btn_add_visit   | ボタン | 訪問記録の入力行を追加                 |
| 2   | 行を追加（Problem）  | btn_add_problem | ボタン | Problem の入力行を追加                 |
| 3   | 行を追加（Plan）     | btn_add_plan    | ボタン | Plan の入力行を追加                    |
| 4   | 削除                 | btn_delete_row  | ボタン | 該当行を削除                           |
| 5   | 下書き保存           | btn_draft       | ボタン | ステータス「下書き」で保存             |
| 6   | 提出                 | btn_submit      | ボタン | ステータス「提出済」で保存、上長へ通知 |

#### バリデーション

| No  | 対象項目 | チェック内容            | エラーメッセージ                           |
| --- | -------- | ----------------------- | ------------------------------------------ |
| 1   | 訪問記録 | 最低 1 件必須（提出時） | 訪問記録を 1 件以上入力してください        |
| 2   | 顧客名   | 必須チェック            | 顧客を選択してください                     |
| 3   | 訪問内容 | 必須チェック            | 訪問内容を入力してください                 |
| 4   | 訪問内容 | 文字数チェック          | 訪問内容は 2000 文字以内で入力してください |

## 4. 共通部品

### ヘッダー

全画面共通で表示。システム名、ログインユーザー名、ログアウトボタンを含む。

### グローバルナビゲーション

ログイン後の全画面で表示。ユーザー権限によって表示項目が変化する。

| メニュー       | 表示条件                       |
| -------------- | ------------------------------ |
| ダッシュボード | 全ユーザー                     |
| 部下日報       | 上長権限を持つユーザー         |
| 顧客マスタ     | 全ユーザー（編集は管理者のみ） |
| 営業マスタ     | 管理者のみ                     |

### ページネーション

一覧画面で使用。1 ページあたり 20 件表示。

### 確認ダイアログ

削除操作、ステータス変更時に表示する共通ダイアログ。
